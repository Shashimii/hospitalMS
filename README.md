
# Hospital Management System

Final Case Study ITEW-4 PNC


## Installation

Clone or Unzip the file

## Backend Setup
```
cd backend
composer install
composer require laravel/sanctum

```
- create .env file mysql as database
- database name "hospitalms"
## Frontend Setup
```
cd frontend
npm install axios
```
